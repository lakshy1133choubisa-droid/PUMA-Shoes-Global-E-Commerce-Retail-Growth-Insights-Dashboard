# PUMA-Shoes-Global-E-Commerce-Retail-Growth-Insights-Dashboard
Designed a full end-to-end Power BI dashboard to analyze PUMA’s global footwear performance across e-commerce and retail channels, integrating key insights across regions, states, cities, retailers, and product lines.

This interactive dashboard transforms 2M+ sales records into dynamic insights that reveal top-selling models, profitability trends, regional opportunities, channel growth, and yearly demand shifts.

Key Contributions:

Built a real-time analytics dashboard covering sales, operating profit, demand, and product performance

Identified top trending PUMA shoes using advanced data modeling (Fast-R Nitro Elite 3, Velocity Nitro 3, Phantom Vision 2 Elite FG)

Analyzed regional market strengths, highlighting top-performing areas such as the West Region (27.7% share)

Created visual insights for sales channels (Outlet, Online & In-Store) showing growth and consumer behavior trends

Developed drill-down views for City, State, Product Name, Retailer ID, and Sales Method

Highlighted seasonal demand spikes and YoY growth trends

Designed a custom Puma-themed UI with branding-aligned color palettes and layout

Business Impact:

Helps leadership quickly identify where PUMA is growing, which models are gaining traction, and what channels are scaling

Supports strategic decisions in inventory planning, retail expansion, e-commerce optimization, and product forecasting

Provides an enterprise-level 360° view of PUMA’s performance for marketing, product, and sales teams

Tools & Skills:
Power BI, DAX, Data Modeling, Dashboard Design, Data Visualization, ETL, Retail Analytics, Trend Analysis, Performance Insights
